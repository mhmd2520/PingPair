Analysis — compare runs

The Analysis tab overlays past sweeps so you can compare them — for example a "before" run against an "after" run, or two different machines on the same LAN.

The Analysis tab — loaded sweeps listed on the left, with their per-case metric lines overlaid on the right for an at-a-glance before / after comparison. Click the image to enlarge.

Loading runs

The tab reads the .json sidecars that every sweep writes. The Source folder defaults to your Save Options destination; point it somewhere else to compare runs saved elsewhere.

  • Refresh — re-scan the source folder for sidecars.
  • Browse… — change the source folder.
  • Add file… — pull in a single sidecar from anywhere on disk (including older one-off files outside the per-sweep layout).

Plotting & comparing

In Loaded runs, tick the runs you want to plot. Each ticked run contributes its per-case metrics to the chart on the right — a single-segment run adds one line, a multi-segment run adds one line per segment.

  • Select all / Uncheck all — toggle every run at once.
  • Rename label — give a run a friendlier name in the legend.
  • Remove — drop a run from the list (doesn't delete the file).

Exporting a comparison

The Export button writes a standalone comparison report (with the overlaid charts) into its own folder next to your other reports — useful for sharing a side-by-side without the raw sweep artefacts.
